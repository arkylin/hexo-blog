在Hexo的_config.yaml里面设置category_map和tags_map
下面这个是例子

```
  {
    学习: Study,
    生活随笔: Life,
    常用配置: common,
    技巧: skill,
    代码: Code,
    安卓开发: android_develop
    }
```
